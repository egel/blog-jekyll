# blog-jekyll
New blog in jekyll without template generation

## install

```bash
rbenv -v
rbenv install -v 2.5.1
rbenv global 2.5.1
ruby -v
gem install bundler jekyll
bundle install
```

## development

```bash
bundle exec jekyll serve --livereload
```
